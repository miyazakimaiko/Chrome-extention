  • merge
    Verb

    Process of taking two or more groups of data and combining the data into a single unified set.

  • attribute

    Noun Adjective

    Specification that defines a property of an object, element, or file. It may also refer to or set the specific value for a given instance of such.

  • argument

    Adverb Adjective

    A value that is passed into a command or a function. For example, if SQR is a routine or function that returns the square of a number, then SQR(4) will return 16. Here, the value 4 is the argument.

  • Variable

    Noun Adjective

    A location which stores temporary data within a program which can be modified, store and display whenever need.
